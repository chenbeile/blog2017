### 我的博客
本着基于学习的目的，我没有选择自己构建博客，而是把在前端项目中遇到的问题，添加到issue里
遇到的问题可以进行分类，也不需要花时间长时间维护博客系统，与我学习前端的思路基本相同
### Issue优点：
1.查找问题更方便，有标签选项
2.编辑简单，使用markdown语法即可，可添加图片文字等
3.有评论类似功能，可以在线交流
